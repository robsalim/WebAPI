# WebAPI - Go микросервис для управления IServer

Микросервис на Go для управления процессом IServer и мониторинга состояния базы данных.

## 📋 Функциональность

- ✅ Мониторинг статуса IServer.exe
- ✅ Перезапуск/остановка IServer.exe
- ✅ Проверка соединения с базой данных (MS SQL Server)
- ✅ Мониторинг задержек данных (>4 часов)
- ✅ Проверка времени через NTP сервер
- ✅ Информация о правах доступа API

## 🚀 Быстрый старт

### Требования
- Go 1.21 или выше
- Windows OS (для управления процессами)
- MS SQL Server (для проверки БД)
- в конфиге прописать путь до SQL, логин/пароль
